## WEB2X API

1. API 구조 및 프로세스

2. API 신청

   1. www.web2x.io에서 ‘로그인' 버튼을 클릭하여, 회원가입과 로그인을 진행합니다.
      ![](/images/1-2-1.png)
   2. ‘프로젝트' 메뉴를 클릭합니다.
      ![](/images/1-2-2.png)
   3. ‘신청하기' 버튼을 클릭합니다.
      ![](/images/1-2-3.png)
   4. ‘회사명'과 ‘상품 사용 용도'를 알맞게 작성합니다.
      ![](/images/1-2-4.png)
   5. ‘신청하기' 버튼을 클릭해 API 사용 신청을 완료합니다.
      ![](/images/1-2-5.png)

   ※ API 사용 신청 후 n일 안에 사용 가능 여부가 결정됩니다.
   ※ 무료 API의 경우 하루 최대 n번까지 요청이 가능합니다. (response 오류가 날 경우, 카운트되지 않음)
